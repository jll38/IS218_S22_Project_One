What is Python?

Python is a high-level programming language with the intent of emphasizing code readability. Python is very popular with machine learning and, in this case, web development. Something very distinctive of Python is that it makes use of indents to determine the flow of code rather than curly brackets.

Flask

Flask is a web-development framework for Python made up of a collection of libraries and modules. Flask's role currently in this project is displaying the HTML files and calling upon Bootstrap V5
If you wish to have a clone of this project on your device, you could type in the following command into any command line that supports git.

Project Files

Here is an outline of this project's filesystem, along with an explanation each file's purpose in this project.

  • Project Root Folder | Root of the Project
    • .github
      • workflows
        • test-build-deploy.yml | Instructions for Github to run the tests build the containers, and deploy to Heroku
    • static
      • images | Stores images for the app
        • Filesystem.PNG | Image of the filesystem as shown on the right
        • me.PNG | A picture of myself used for testing
    • templates | Stores HTML files
      • docker.html | Docker Webpage
      • flask.html | Scrapped Webpage (Combined with python.html)
      • git.html | Git Webpage
      • index.html | Homepage
      • python.html | Python and Flask Webpage (Current Page)
    • tests | Contains testing files
      • pytest.py | Testing file for Github
    • venv | Library Root Dir
    • .dockerignore | Tells docker what to ignore
    • .gitignore | Tells Git what to ignore
    • app.py | Initializes app and renders HTML pages
    • docker-compose.yml | Orchestration file that contains the confirguration of the project.
    • Dockerfile | File used to create an image to run inside a container that runs the program
    • Heroku.yml | Configures Heroku deployment
    • requirements.txt | list of requirements that Python installs
